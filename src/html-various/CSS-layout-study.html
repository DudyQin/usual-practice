注解：clear只会清除source document中在它之前的浮动盒子，不会清除source document中在它之后浮动盒子，如下，实际上source document中在它之后的浮动盒子本身也就对block-level的元素没有影响(另外注意只有block-level的元素才有清除浮动的功能)
